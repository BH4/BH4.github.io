About

I'm a computational physicist and machine learning researcher passionate about solving complex problems with code, math, and creativity.

I began programming early on, teaching myself Python to tackle challenging problems, and that curiosity grew into a career at the intersection of machine learning, high-performance computing, and physics.

Over the past few years, I’ve:

  • Designed and trained neural architectures to represent complex scientific state spaces, using sampling-based optimization and scalable HPC pipelines instead of traditional supervised datasets.
  • Optimized large-scale simulations to run efficiently on DOE supercomputers, reducing bottlenecks and improving throughput for multi-node workloads.
  • Developed open-source automation tools ranging from automated literature searches to an advanced auto-clicker.

While my research has focused on nuclear physics, the core of my work is solving hard technical problems—whether that means developing scalable machine learning models, optimizing complex simulations, or building tools that make advanced computing workflows more accessible.
