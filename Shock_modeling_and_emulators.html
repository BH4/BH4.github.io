Shock physics at Los Alamos National Lab

In graduate school I resolved to combine my interest in physics and machine learning by attending a summer school at Los Alamos National Laboratory. In addition to quickly learning about a branch of physics I hadn't previously interacted with, the machine learning aspects of the summer project included:

  • Developed and implemented machine learning algorithms to analyze shock-driven ejecta using the Keras API for TensorFlow.
  • Used clustering techniques, including k-means, for unsupervised learning of data features.
  • Implemented deep learning models to emulate results from high fidelity simulations.

As I had hoped, combining my interests in physics problem solving with my passion for machine learning and algorithms led to fascinating work. Subsequently, I was able to find my current position at Argonne working on neural network quantum states.
